            Requirements

            To use this app, your IAM user must have permissions for ListBucket, DeleteObject, GetObject, and PutObject on the target bucket, and the S3 bucket must have CORS enabled to allow browser access.

            IAM Policy

            {
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "AllowListBucket",
            "Effect": "Allow",
            "Action": [
                "s3:ListBucket",
                "s3:GetObject",
                "s3:PutObject",
                "s3:DeleteObject"
            ],
            "Resource": "arn:aws:s3:::BUCKET-NAME"
        }
    ]
}
